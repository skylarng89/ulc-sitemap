A fast and secure XML sitemap generator for WordPress with advanced caching and customization options.

== Description ==

ULC Sitemap is a high-performance XML sitemap generator designed for WordPress websites of any size. It automatically generates and updates your sitemap while providing full control over included content and priorities.

= Features =

* Automatic sitemap generation
* Advanced caching system for optimal performance
* Support for posts, pages, custom post types, categories, tags, and author archives
* Customizable priorities and change frequencies
* Clean, valid XML output
* Automatic ping to search engines when sitemap is updated
* SEO-friendly URL structure
* Multi-language support
* WordPress REST API integration

= Performance =

* Efficient caching system
* Minimal database queries
* Optimized for large websites
* Low memory footprint

= Developer Friendly =

* Extensive hooks and filters
* Well-documented code
* Unit tested
* PSR-12 compliant
* Modern PHP 8.2+ features

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ulc-sitemap`, or install through WordPress plugins screen
2. Activate the plugin
3. Go to Settings > ULC Sitemap to configure options

== Frequently Asked Questions ==

= How often is the sitemap updated? =

By default, the sitemap is updated daily or whenever you publish/update content.

= Can I exclude certain content types? =

Yes, you can exclude any post type, taxonomy, or author archive through the settings page.

= Is it compatible with my SEO plugin? =

Yes, ULC Sitemap works alongside popular SEO plugins without conflicts.
